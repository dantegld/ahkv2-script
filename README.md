# AHK V2 script
## Features

### Disable Microsoft Copilot key

>For laptops with Microsoft Copilot key

Disables the key and rebinds it to ```RCtrl```

### Vim-Style Navigation

> Commented out right now since its not finished.

Use ```RCtrl + HJKL``` for arrow keys.  
Use ```RCtrl + UI``` for ```Home``` and ```End```.  
Use ```RCtrl + M``` for ```Backspace```.  


### Win + T toggles touchscreen

>Only tested on Asus touchscreen.

Disables default ```Win + T``` functionality (Navigating taskbar icons).

### Win + B opens default browser
Disables default ```Win+ B``` functionality (Focussing the notification area)

### Win + W closes focused application
Just runs ```WinClose("window")``` (```alt+f4```)
